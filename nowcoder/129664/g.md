

ymyb队长邀请了 n 位acm的同学参加团建。同学们坐在一个圆桌周围。对于所有 i，从 1 到 n，同学i 坐在同学 (imodn)+1 和 ((i+n−2)modn)+1 旁边。

ymyb队长准备了 n 道菜。桌子上有 n 个位置。位置 i 在同学 i 的前面。同学 i 只能接触到放在位置 i、(imodn)+1 和 ((i+n−2)modn)+1 处的菜。ymyb队长将在每个位置上放置一道菜。

在这些菜中，有 a 道是辣的，n−a 道是不辣的。有些（可能为 0）同学不能吃辣的食物。如果一位同学可以吃辣的食物，他/她的满意度水平是他/她可以接触到的菜的数量（无论是辣的还是不辣的）。如果一位同学不能吃辣的食物，他/她的满意度水平是他/她可以接触到的不辣的菜的数量。

ymyb队长知道每位同学是否可以吃辣的食物。请帮助他安排桌子上的菜，使得所有同学的满意度水平之和最大化。输出最大的总和。

输入描述:
第一行包含两个整数 n,a (3≤n≤105,0≤a≤n)。

第二行包含 n 个整数 b1,…,bn。bi 是 0 或 1。bi=1 表示同学i 可以吃辣的食物。bi=0 表示同学 i 不能吃辣的食物。

输出描述:
输出一行一个整数，表示答案。

示例1
输入
复制
5 2
1 0 1 0 1
输出
复制
13
