# F 前缀和
（单纯恶心人的题目）
n * m <= 1e5
我们让 n 是小的， m 是大的，就可以用 O(n^2*m) 的做法
1. 枚举行的起点和终点，纵向前缀和统计每一列的和（将二维转换成一维）
2. 对于每个行起点、终点的组合，按顺序统计横向前缀和相同的，计算贡献 / 或者统计前缀和的出现次数，算组合数C(cnt, 2)
